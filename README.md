# YouTube Videos Listing UI

A responsive YouTube-style video listing interface built using React, Vite, Tailwind CSS, and FreeAPI.

## Features

- Fetch videos using FreeAPI
- Responsive video grid layout
- Modern YouTube-style UI
- Loading state handling
- Reusable React components
- Hover effects and clean design

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## API Used

https://api.freeapi.app/api/v1/public/youtube/videos
